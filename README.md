# STEM Resource Access Hub

A centralized web platform designed to streamline access to university STEM resources. This system allows students to browse and reserve laboratories and equipment online, while administrators can manage bookings, track usage, monitor safety compliance, and optimize resource allocation through real-time analytics.

---

## üöÄ Features

### For Students
- Browse available laboratories and equipment.
- Reserve resources online without manual paperwork.
- View booking statuses and avoid double-booking conflicts.

### For Administrators
- Dashboard to monitor bookings and resource usage.
- Track equipment, lab availability, and safety compliance.
- Real-time analytics for better resource allocation.
- Automated booking validation to prevent scheduling errors.

### Problems Solved
- Simplifies access to STEM labs and equipment.
- Eliminates double-booking and scheduling mistakes.
- Ensures fair distribution of limited resources.
